# 📚 Sistema de Alunos em Java

Projeto desenvolvido para prática de lógica de programação e conceitos básicos de Java.

## 🚀 Funcionalidades

* Cadastro de alunos
* Cálculo de média
* Classificação (Aprovado, Recuperação, Reprovado)
* Validação de dados
* Geração de relatório em arquivo

## 🛠️ Tecnologias

* Java
* Programação Orientada a Objetos
* Enum
* Manipulação de arquivos

## 📌 Aprendizados

* Estruturas condicionais (if/else)
* Laços de repetição (for, while, do-while)
* Listas (ArrayList)
* Escrita em arquivo (BufferedWriter)
* Organização em pacotes

## 📁 Estrutura

* application
* model.entities
* model.enums
