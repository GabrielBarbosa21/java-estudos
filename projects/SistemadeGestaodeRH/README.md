📌 Sistema de Funcionários em Java

Projeto desenvolvido com o objetivo de praticar conceitos fundamentais de Java e Programação Orientada a Objetos (POO).

🚀 Funcionalidades
Cadastro de funcionários
Validação de ID único
Atualização de salário por porcentagem
Listagem de funcionários
Identificação do funcionário com mais tempo de empresa
🛠️ Tecnologias e conceitos utilizados
Java
Programação Orientada a Objetos
Encapsulamento
Listas (List)
Stream API
Manipulação de datas (LocalDate)
Cálculo de tempo com ChronoUnit
